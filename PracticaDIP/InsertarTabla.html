Insertar tabla
Para realizar una tabla, primero se usa la etiqueta table para definirla, luego con la etiqueta tr realizamos cada fila de una tabla.
Para crear un encabezado en la tabla, se usa la etiqueta th, estas se distinguen visualmente ya que por defecto, el contenido de las celdas de encabezado se muestra en negrita y centrado
Para luego, crear las celdas de los datos usando la etiqueta td, el uso de las tablas ayuda para poder organizar datos dentro de una pagina web.
