Tim Berners-Lee es un científico de la computación británico conocido por inventar la World Wide Web en 1989 mientras trabajaba en CERN. Su visión era crear un sistema de información global que permitiera a los investigadores compartir datos de manera eficiente.

Inventó el lenguaje HTML, el protocolo HTTP y el primer navegador web. También fundó el World Wide Web Consortium (W3C) para desarrollar estándares web abiertos.
